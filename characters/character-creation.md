# Character Creation

Start with:
- Crucible & Fire (both rating **2**)
- **3** trait (up to **3**) with rating **1**
- **1** in each attribute

**15** points

**1** point:
- [9x] **+1** level in a skill
- [4x] **+1** in an attribute
- [1x] **+1** rating of a trait
